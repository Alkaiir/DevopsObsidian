### Bash скрипт -
файл содержащий список команд для выполнения в linux.
### Знакомство и навигация

В linux нет дисков D,C. Тут всё начинается с корня ``` /``` . В нём находятся все папки, диски и даже флешки.

Для навигации и перемещения из корня по системе используются следующие команды.

=================================================

```pwd``` - print working directory - показывает полный путь до места, где ты сейчас находишься

![[Pasted image 20260401000828.png]]

=================================================

```ls``` - list - показывает список файлов и папок в текущей директории
	```ls -l``` - покажет подробности (размер , дата)
	```ls -a``` - покажет скрытые файлы
	
![[Pasted image 20260401001200.png]]

![[Pasted image 20260401001228.png]]

=================================================

```cd``` - команда для перемещения
```cd Documents``` - зайти в папку Documents
```cd ..```  - выйти на один уровень вверх (назад)
```cd ~``` - вернуться в домашнюю директорию

![[Pasted image 20260401001721.png]]

=================================================

```mkdir <имя директории>``` - создание директории
```touch <имя файла>``` - создание файла

=================================================

```rmdir <имя директории>``` - удаление  директории
```rm <имя файла>``` - удаление файла

Для удаления директории с файлами и вообщем принудительного удаления используется флаг -f  (force).

=================================================

```cp <путь к файлу> <путь куда скопировать>``` - копирование файла (для копирования папки с содержимым используется флаг  -r)

```mv <путь к файлу> <путь куда переместить>``` - перемещение и переименование файла
![[Pasted image 20260401004511.png]]

=================================================

```man``` - команда для просмотра информации о команде

пример:
```man cd```
